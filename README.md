# Face Detection

This Python project execute commands to open the webcam.


## Project Description

- Python3 
- Computer Vision -> openCV
 


  
## Installation / Usage

Install face-detection with pip

```bash
  pip install requirements.txt
  cd face-detection
  python face_eye.py
```
